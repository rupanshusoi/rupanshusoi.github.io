Dynamic Analysis for Index Launches

[Code]
I developed a dynamic program analysis for determining the correctness of applying an optimization, known as an index launch, to a certain class of loops in the Regent programming language. To confirm that the analysis had negligible run-time performance impact, I ran several experiments on Piz Daint, the #12 supercomputer in the world.

Selective Repeat Inspired File Transfer Protocol in Racket

[Code]
For my computer networks course, I had to design and implement a reliable file transfer protocol over UDP sockets. To challenge myself, I decided to use a language I had never used before: Racket. My implementation is able to handle a variety of adverse network conditions: packet loss, delays, reordering, and corruption, and achieve speeds of upto 2.6 Gbps on local transfers.

Runi: Handwritten Lexer and Parser in Go

[Code]
Runi is a context-free grammar, lexer, and recursive descent parser for a C-like language written in Go, with support for generating parse tree visualizations (depicting the leftmost derivation) using Graphviz.

Brendr: Efficient Borrowing-Lending for Close-Knit Communities

[Code] [Prototype]
Brendr is an online platform for borrowing-lending in college campuses (and similar closely knit communities) implemented using the MERN stack. It is a feature of Brendr to not support monetary transactions. A working prototype is hosted on Heroku.

An Implicitly Parallel Meshfree Solver in Regent

[Code] [Abstract] [Paper] [Slides]
I developed a high-performance CFD solver based on q-LSKUM in Regent. Regent is an implicitly parallel programming language, which means users can write sequential code which is parallelized automatically. I compared my implementation against Fortran and Julia and published the results as an empirical study. I was fortunate to be part of Dr. Anil Nemili's scientific computing research group for this project.
